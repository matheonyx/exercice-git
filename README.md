# Exercice de Collaboration Git & GitHub

Dépôt réalisé par **Mathéo (matheonyx)** et **Louis (Loukilo27)**.

## Objectifs validés

### 1. Exercice Github/clone
* Dépôt cloné par le collaborateur.
* Ajout du nom du collaborateur dans `index.html`.
* Push sur le master distant.
* Pull du propriétaire pour synchronisation.

### 2. Exercice Github/fetch
* Création d'une branche `contrib` sur le clone.
* Réalisation de deux commits sur cette branche.
* Récupération des modifications du master via `git fetch`.
* Merge de la branche de contribution dans le master local.
* Push final vers le dépôt distant.

## Graphe de l'historique
Commande utilisée pour la visualisation :
`git log --oneline --all --graph --format="%h |%d| - %an - (%s) [%cd]"`

## Gestion des conflits
* Résolution manuelle d'un conflit de fusion sur `index.html` lors de la synchronisation des travaux entre Mathéo et Louis.
